College Network Topology Implementation

IC10050 - Networking Project

Designed and implemented a comprehensive college network infrastructure using Cisco Packet Tracer. The project involved creating a multi-departmental network topology spanning IT, Office, Principal's Room, Internet Lab, Server Room, and Computer Department. Configured VLANs to segregate traffic, implemented DHCP for automatic IP assignment, and set up DNS, FTP, and web servers. The network includes proper routing protocols (OSPF/EIGRP) and switching techniques to ensure redundancy and efficient traffic flow.

Key Achievements: Successfully configured 30+ networked devices including routers, switches, PCs, printers, and servers. Implemented comprehensive IP addressing scheme with proper subnet masks and gateway configurations. Created detailed network documentation including topology diagrams and device configurations.

Cisco Packet Tracer Network Design VLANs DHCP/DNS OSPF Routing Network Security

Hospital Operation Scheduler

IC10039 - Team Project

Developed a sophisticated hospital operation scheduling system as part of a team project. The application manages electronic diaries for healthcare professionals (doctors, nurses, physiotherapists) and schedules patient appointments efficiently. Implemented advanced features including appointment searching across multiple professional diaries, conflict detection, and undo functionality for diary modifications.

Technical Highlights: Built using Java with custom data structures optimized for appointment searching. Implemented file I/O for persistent storage of diary entries. The search algorithm efficiently identifies available time slots considering multiple professionals' schedules. Included performance metrics to measure search operation speed.

Java Data Structures Algorithms File I/O Team Development Software Design
